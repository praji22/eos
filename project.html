Semantic Network builder

Our highly ambitious project aimed to enable users to get the best out of their existing social networks (currently integration with facebook, orkut), to build dynamic networks / n-dimensions (using existing Web2.0 systems / data). The systems semantically ties in various user profile information, their connections (n-degree), communities to dynamically create a network defined by user's perspective. For e.g., create a network of people with similar interests / hobbies. Based on Java / J2EE, AJAX, Javascript, Semantic technologies – OWL, RDFS, Jena, SPARQL, Oracle triple store, etc.
